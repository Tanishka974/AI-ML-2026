## Projects

### 1. ML1_insurance.ipynb
**Linear Regression Model for Insurance Charge Prediction**

This was one of my initial machine learning models where I used **linear regression** to predict insurance charges based on various customer features.

#### Features Used:
- Age
- BMI (Body Mass Index)
- Smoker status
- Gender
- Number of children
- Region

#### Project Workflow:
1. **Data Cleaning** - Cleaned and prepared the dataset for analysis
2. **Categorical Variable Encoding** - Converted categorical variables (gender, region, smoker status) into numerical format
3. **Feature Engineering** - Built interaction features to capture relationships between variables
4. **Train-Test Split** - Divided data into training and testing sets for proper model evaluation
5. **Model Training** - Trained a linear regression model on the training data
6. **Model Evaluation** - Evaluated performance using metrics like R² (coefficient of determination)

#### Key Learnings:
- Understanding regression fundamentals and how models make predictions
- How different features impact real-world financial predictions
- Importance of data preprocessing and feature engineering
- Model evaluation and performance metrics interpretation

#### Technologies Used:
- Python
- Pandas (data manipulation)
- Scikit-learn (machine learning)
- Matplotlib/Seaborn (visualization)
