# Project Explanations

## ML1_insurance.ipynb
This project focuses on predicting insurance premiums based on various features such as age, sex, body mass index (BMI), number of children, and whether the individual is a smoker. The goal is to develop a regression model that can effectively estimate the premium costs for different profile types.

## ML2_heartattack_pred.ipynb
This project aims to predict the likelihood of a heart attack based on medical data. It utilizes classification algorithms to assess risk factors including age, cholesterol levels, blood pressure, and other health indicators to provide timely insights that can be used for preventive healthcare measures.
